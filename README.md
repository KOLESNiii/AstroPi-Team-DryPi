# AstroPi-Team-DryPi
Team DryPi's AstroPi 2023 Submission- finding the change in desertification around the globe

Modules we can use:
  - Skyfield
  - picamera
  - colorzero
  - GPIO Zero
  - GDAL
  - GeoPandas
  - EarthPy
  - NumPy
  - SciPy
  - TF Lite
  - PyCoral
  - pandas
  - logzero
  - Matplotlib
  - Pillow
  - OpenCV
  - exif
  - scikit-learn
  - scikit-image
  - reverse-geocoder
  - sense_hat
  - pisense
